# PESTLE and SIPOC

This project visualizes PESTLE using the 'manim' library. The script demonstrates the relationship between customer satisfaction and feature functionality.

## How to Run 

1. Install the required dependencies using 'requirements.txt'.
2. Use a Conda environment to run the script.
3. Execute 'PESTLE.py' using Manim;
   manim PESTLE.py
4. to see the animation of pestleresp.py use manim-pql pestleresp.py PESTLEResponsible.py
5. Execute 'pestleresp.py using Manim 

## Files

- 'PESTLE.py: main script.
- 'requirements.txt': List of dependencies.
- 'README.md': Project documentation.
